# TetraNova 俄罗斯方块

一个基于 `Pygame` 开发的俄罗斯方块项目，支持登录注册、排行榜、无尽模式、限时模式、双人对战和人机对战（AI）。

## 功能特性

- 多游戏模式：无尽、限时（180 秒）、双人对战、人机对战
- 账号系统：本地注册/登录（SQLite）
- 排行榜：记录并展示历史高分
- 道具系统（单人模式）：炸弹、闪电、垃圾桶
- 音效与背景音乐：支持开关
- 可视化设置：支持网格显示开关

## 运行环境

- Python 3.8+
- 主要依赖：
  - `pygame`
- 可选依赖（仅 `test.py` 可能使用）：
  - `Pillow`
  - `matplotlib`

安装依赖示例：

```bash
pip install pygame pillow matplotlib
```

## 快速开始

在项目根目录执行：

```bash
python game.py
```

首次运行会自动在 `data/tetris.db` 创建本地数据库文件。

## 游戏模式说明

- 无尽模式：无时间限制，直到失败
- 限时模式：180 秒内尽可能拿高分
- 双人模式：两位玩家对战，支持垃圾行机制
- 人机模式：左侧 AI、右侧玩家

## 按键说明

### 单人模式

- 移动：`←` `→` 或 `A` `D`
- 软降：`↓` 或 `S`
- 旋转：`↑` 或 `W`
- 硬降：`Space`
- 暂停：`P`
- 暂停后返回菜单：`M`

### 双人模式

- 玩家1（左）：
  - 移动：`A` `D`
  - 软降：`S`
  - 旋转：`W`
  - 硬降：`Space`
  - 暂停：`P`
- 玩家2（右）：
  - 移动：`←` `→`
  - 软降：`↓`
  - 旋转：`↑`
  - 硬降：`Enter`
  - 暂停：`O`
- 返回菜单：暂停状态下按 `M`

### 人机模式

- AI 自动控制左侧玩家
- 你控制右侧玩家（与双人模式玩家2一致）：
  - `←` `→` `↓` `↑` `Enter`
  - 暂停：`O`
- 返回菜单：暂停状态下按 `M`

### 通用按键

- 游戏结束后：
  - 重新开始：`R`
  - 返回菜单：`M`
- 帮助/关于页面返回：`B` 或 `Esc`

## 道具系统（单人）

- 炸弹：消除最下面一行（200 分）
- 闪电：清空当前棋盘（1000 分）
- 垃圾桶：跳过当前方块（100 分）

## 数据与目录说明

- 主程序：`game.py`
- 棋盘逻辑：`board.py`
- UI 绘制：`ui.py`
- 双人/人机逻辑：`two_player.py`、`ai_game.py`、`ai_player.py`
- 数据库：`database.py`（SQLite，文件在 `data/tetris.db`）
- 音效管理：`sound_manager.py`
- 字体管理：`font_manager.py`

## 资源文件要求

代码中默认会读取以下资源目录，请确保存在并包含对应文件：

- `assets/images/`（如图标、道具图片）
- `assets/sounds/`（点击、消行、BGM 等音频）
- `assets/fonts/`（如 `msyh.ttc`）

若资源缺失，程序启动或运行到对应功能时会报错。
